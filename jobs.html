Background Jobs Overview

The FinLocker Analytics system includes several background jobs that handle data processing, synchronization, and reporting tasks.

Available Jobs

Loan Officer Reports

Generates comprehensive performance reports for loan officers, including metrics on client interactions, loan pipeline status, and productivity analytics.

Supporting Data Job

Processes and synchronizes supporting data required for analytics calculations, ensuring all reference data is up-to-date.

Data Sync Job

Handles synchronization of data between different systems and databases, maintaining data consistency across the platform.

User Attributes Job

Processes user attribute data and behavioral analytics, updating user profiles and engagement metrics.

Job Scheduling

All jobs are designed to run as background processes and can be scheduled using:

  • Azure Functions Timer Triggers
  • Hangfire recurring jobs
  • Windows Task Scheduler
  • Cron jobs (Linux environments)

For detailed implementation and configuration, refer to each job's specific documentation.
