FinLocker GenAI

Welcome to the FinLocker GenAI documentation.

Overview

This project contains GenAI (Generative AI) services and applications for the FinLocker platform, providing comprehensive AI-powered features including content generation, evaluation, market reporting, and intelligent data processing capabilities.

Key Components

Core Libraries

  • FinLocker.GenAI.Application: Core application layer with business logic and AI services
  • FinLocker.GenAI.Contracts: Contracts and interfaces for queue messaging and API communication
  • FinLocker.GenAI.Database: Database layer with Cosmos DB contexts and models
  • FinLocker.GenAI.Types: Shared types, enums, and data models

API Services

  • FinLocker.GenAI.Customer.Api: Customer-facing REST API service
  • FinLocker.GenAI.Consumer.Api: Consumer REST API service
  • FinLocker.GenAI.Internal.Api: Internal REST API service

Background Processing

  • FinLocker.GenAI.Evaluator.Queue: AI evaluation and scoring queue worker
  • FinLocker.GenAI.Runner.Queue: AI content generation queue worker

Background Jobs

  • FinLocker.GenAI.MarketReport.SyncJob: Synchronizes and generates market reports using AI

Development

Prerequisites

  • .NET 8.0 SDK
  • Azure Cosmos DB
  • Azure Service Bus (for queue processing)
  • OpenAI or Azure OpenAI API access

Key Technologies

  • Entity Framework Core: Database access with Cosmos DB provider
  • Azure Service Bus: Message queuing for AI processing
  • OpenAI APIs: AI content generation and evaluation
  • ASP.NET Core: Web API framework
  • Azure Cosmos DB: NoSQL database for AI data storage
